 RANDOM PASSWORD GENERATOR

-->PROJECT DESCRIPTION

This project is a simple Random Password Generator developed using Python.It generates a secure password based on the length provided by the user through the command line.

-->OBJECTIVE

To create a beginner-level Python application that generates random passwords using letters, numbers,and special characters.

-->TECHNOLOGIES USED

.Python 3
.Built-in modules
  .random
  .string

-->FEATURES

.User-defined password length
.Random password generation
.Includes
  .Alphabets (uppercase & lowercase)
  .Numbers
  .Special characters
.Input validation using error handling

-->HOW TO RUN THE PROGRAM

.Make sure Python 3 is installed
.Save the file as
   password_generator.py
.Open Terminal or VS code
.Run the program
   python password_generator.py
.Enter the password length when prompted
